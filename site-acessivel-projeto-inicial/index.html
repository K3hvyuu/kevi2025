O que é a dança espontânea?

A Dança espontânea é um estilo de dança que se caracteriza por movimentos livres e instintivos, sem coreografia. É uma forma de expressão que transmite uma mensagem.

No louvor

A dança espontânea no louvor é uma adoração a Deus, onde cada pessoa se entrega ao Senhor através de seus movimentos. É uma forma de ministração onde o ministro se deixa mover pelo Espírito Santo. A dança espontânea no louvor é diferente de uma coreografia, onde os passos são sincronizados.
